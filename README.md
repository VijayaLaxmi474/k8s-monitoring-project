# Kubernetes Monitoring Project

## Overview

This project demonstrates monitoring a Kubernetes application using Prometheus and Grafana.

## Technologies

- Docker
- Kubernetes
- Minikube
- Prometheus
- Grafana
- Python Flask

## Architecture

Flask App
↓
Kubernetes Deployment
↓
Prometheus
↓
Grafana Dashboard

## Features

- Kubernetes Deployment
- NodePort Service
- Prometheus Metrics Collection
- Grafana Visualization

## Commands

kubectl get pods
kubectl get services
kubectl get deployments

## Outcome

Successfully monitored Kubernetes workloads using Prometheus and Grafana.
